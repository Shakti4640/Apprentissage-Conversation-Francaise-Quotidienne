🟡 PROJECT 24: "It Was Done, It Was Built"

You're at a museum, reading news, hearing formal announcements — things were done BY someone

🎯 THE HYPOTHETICAL PROBLEM

→ You're visiting a museum in Paris with a French friend
→ You read plaques, listen to audio guides, overhear tour guides speaking
→ Every sentence sounds backwards compared to what you've learned:

Museum plaque:  "Ce tableau A ÉTÉ PEINT par Monet en 1872."
Audio guide:    "Le château A ÉTÉ CONSTRUIT au XIIe siècle."
Tour guide:     "Cette salle EST UTILISÉE pour les expositions temporaires."

→ You turn on the news that evening:

News anchor:    "Le suspect A ÉTÉ ARRÊTÉ par la police ce matin."
Weather:        "Des pluies SONT ATTENDUES dans le sud."
Politics:       "La loi A ÉTÉ VOTÉE par l'Assemblée nationale."

→ You understand the individual words — but the sentence architecture is unfamiliar
→ In every sentence you've learned so far, the DOER comes first:

ACTIVE (what you know):
→ Monet A PEINT ce tableau.        (Monet painted this painting.)
→ La police A ARRÊTÉ le suspect.   (The police arrested the suspect.)

→ But in these new sentences, the RECEIVER comes first — and the doer comes last (or disappears entirely):

PASSIVE (what you're hearing):
→ Ce tableau A ÉTÉ PEINT par Monet.     (This painting was painted by Monet.)
→ Le suspect A ÉTÉ ARRÊTÉ par la police. (The suspect was arrested by the police.)

→ This is the PASSIVE VOICE — the sentence flips so the OBJECT becomes the SUBJECT
→ French uses it heavily in formal speech, news, museums, official documents, and storytelling
→ Without understanding it, you miss half of what you read and hear in formal French

The core realization:
→ All previous projects built active sentences — Subject DOES something to Object
→ Project 24 builds passive sentences — Object HAS something done to it BY Subject
→ The passive voice uses ÊTRE + PARTICIPE PASSÉ — the same structure as passé composé with être (Project 9)
→ But here, ÊTRE carries the TENSE — and the participe passé carries the ACTION
→ This creates a tense-flexible system — passive works in present, past, future, and beyond

🔍 HOW TO APPROACH IT

→ Every passive sentence follows this skeleton:

SUBJECT (receiver) + ÊTRE (conjugated in any tense) + PARTICIPE PASSÉ (agrees) + PAR + AGENT (doer)

→ The agent (par + doer) is OPTIONAL — often omitted when the doer is unknown or unimportant

→ Think of it as a three-part transformation:

ACTIVE:   [DOER] + [VERB] + [RECEIVER]
          La police  a arrêté  le suspect.

PASSIVE:  [RECEIVER] + [ÊTRE + PP] + [PAR + DOER]
          Le suspect  a été arrêté  par la police.

→ ÊTRE takes the tense (present, passé composé, imparfait, futur, conditionnel, etc.)
→ PARTICIPE PASSÉ agrees with the NEW SUBJECT (the receiver) in gender and number
→ PAR introduces the original doer — but can be removed

🔧 HOW CAN WE SOLVE IT

Step 1 — The Tense System for Passive Voice

TenseÊTRE FormFull Passive StructureExample
Présentest / sontest + PPLa porte est fermée.
Passé composéa été / ont étéa été + PPLa porte a été fermée.
Imparfaitétait / étaientétait + PPLa porte était fermée.
Futur simplesera / serontsera + PPLa porte sera fermée.
Futur procheva êtreva être + PPLa porte va être fermée.

→ The participe passé ALWAYS agrees with the subject in passive voice — no exceptions
→ La porte est fermée (fem sing)
→ Les livres sont vendus (masc plural)
→ Les lettres ont été envoyées (fem plural)

Step 2 — Agreement Rules (Non-Negotiable)

The past participle in passive voice ALWAYS agrees with the SUBJECT:

MASCULINE SINGULAR:    Le livre est vendu.         (+nothing)
FEMININE SINGULAR:     La maison est vendue.       (+e)
MASCULINE PLURAL:      Les livres sont vendus.     (+s)
FEMININE PLURAL:       Les maisons sont vendues.   (+es)

This is AUTOMATIC — no exceptions — because the auxiliary is ÊTRE
(Same rule as Project 9 — être verbs agree with subject)

Step 3 — PAR vs. DE — Two Ways to Introduce the Agent

PAR = by (for ACTIONS — the most common)
→ Le gâteau a été fait PAR ma mère.

DE = by (for STATES, EMOTIONS, PERCEPTIONS — less common)
→ Elle est aimée DE tout le monde. (She is loved by everyone.)
→ Il est respecté DE ses collègues. (He is respected by his colleagues.)
→ La maison est entourée DE arbres. (The house is surrounded by trees.)

RULE OF THUMB:
→ Physical/concrete action → PAR (built, broken, arrested, sold, written)
→ Emotional state/perception → DE (loved, respected, known, surrounded, covered)
→ When in doubt → PAR (always safe)

🧠 CRITICAL OBSERVATION — Passive Present vs. State Description:

"La porte est fermée" can mean TWO things:
1. The door IS closed (state — description)
2. The door IS BEING closed (passive action — happening now)

Context tells you which:
→ "La porte est fermée à clé." = The door is locked. (STATE)
→ "La porte est fermée par le gardien chaque soir." = The door is closed by the guard every evening. (ACTION)

When there's an AGENT (par...) → it's clearly PASSIVE ACTION
When there's no agent → it could be either — context decides

🧠 CRITICAL RULE — VERBS THAT CANNOT BE PASSIVE IN FRENCH:

In English, you can make almost ANY verb passive.
In French, ONLY VERBS WITH A DIRECT OBJECT (COD) can be made passive.

INDIRECT OBJECT verbs (COI — verbs with "à") CANNOT be passive:

❌ "She was told" → dire À quelqu'un → CANNOT be passive
   ✅ On lui a dit.

❌ "He was given" → donner À quelqu'un → CANNOT be passive
   ✅ On lui a donné.

❌ "They were asked" → demander À quelqu'un → CANNOT be passive
   ✅ On leur a demandé.

❌ "I was offered" → offrir À quelqu'un → CANNOT be passive
   ✅ On m'a offert.

RULE: If the person is an INDIRECT OBJECT (COI) in active voice
→ They CANNOT become the subject of a passive sentence
→ Use "ON" + active voice instead

🧠 "ON" — THE FRENCH PASSIVE REPLACEMENT:

French VERY often uses "ON" + active verb instead of passive voice:

PASSIVE:  Le français est parlé ici.
ACTIVE:   On parle français ici.        ← MORE NATURAL in speech

PASSIVE:  Le suspect a été arrêté.
ACTIVE:   On a arrêté le suspect.       ← MORE NATURAL in casual contexts

"ON" = unspecified agent = "someone" / "people" / "they" / "we"
→ This is the PREFERRED structure in spoken French
→ Passive voice is more common in WRITTEN / FORMAL French

🔑 KEY TAKEAWAY

The passive voice flips your sentence perspective — from WHO did it to WHAT was done.

→ Projects 1-23 built active sentences — the doer is always the subject
→ Project 24 introduces passive sentences — the receiver becomes the subject

→ The system is elegantly simple:

ComponentRole
ÊTRE (any tense)Carries the TIME of the action
PARTICIPE PASSÉCarries the ACTION (always agrees with subject)
PAR + agentIdentifies WHO did it (optional)

→ Three ways to express "someone did something" in French:

StyleExampleBest For
ActiveLa police a arrêté le suspect.Casual, personal, storytelling
PassiveLe suspect a été arrêté par la police.Formal, news, official, written
ON + activeOn a arrêté le suspect.Casual, unknown agent

→ You can now read news, understand museum plaques, follow formal announcements
→ The passive voice is your gateway to understanding formal French

→ But what happens when someone TELLS you what another person said?
→ "Il a dit qu'il était fatigué." — that "était" is not imparfait in the usual sense
→ It's reported speech — the tenses SHIFT when you report what someone else said
