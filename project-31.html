→ Subjonctif isn't just for emotions and obligations
→ It appears after purpose connectors (pour que, afin que)

→ It appears after superlatives and unique/restrictive expressions

→ And it has a PAST form (subjonctif passé) for completed actions
→ Without full mastery, you'll hit a ceiling — you'll understand 80% of French but miss the sophisticated 20%

NEW — THE SUBJONCTIF PASSÉ

FORMATION: AVOIR or ÊTRE (in subjonctif présent) + PARTICIPE PASSÉ

AVOIR verbs:
que j'aie mangé          que nous ayons mangé
que tu aies mangé         que vous ayez mangé
qu'il ait mangé           qu'ils aient mangé

ÊTRE verbs:
que je sois arrivé(e)     que nous soyons arrivé(e)s
que tu sois arrivé(e)     que vous soyez arrivé(e)(s)
qu'il soit arrivé         qu'ils soient arrivés
qu'elle soit arrivée      qu'elles soient arrivées

REFLEXIVE verbs:
que je me sois reposé(e)
qu'il se soit trompé
qu'elle se soit souvenue

WHEN TO USE:
→ When the action in the subjonctif clause happened BEFORE the main clause
→ "Je suis content que tu SOIS VENU." = I'm happy you came (you came BEFORE I'm happy)
→ "C'est dommage qu'il AIT PERDU." = It's a shame he lost (he lost BEFORE it's a shame)

COMPARISON:
→ Subjonctif présent: action simultaneous or future relative to main clause
   "Je veux qu'il VIENNE." = I want him to come (hasn't happened yet)
→ Subjonctif passé: action COMPLETED before main clause
   "Je suis content qu'il SOIT VENU." = I'm happy he came (already happened)

🧠 "POUR QUE" vs. "AFIN QUE" vs. "POUR" + infinitif:

POUR QUE + subjonctif = So that [someone else] does X
→ DIFFERENT subjects required
→ "Je t'explique POUR QUE TU comprennes." (I explain / you understand)
→ Register: Universal — very common in speech and writing

AFIN QUE + subjonctif = So that [someone else] does X
→ Same meaning as "pour que" — MORE FORMAL
→ "Afin que vous compreniez..." = So that you understand...
→ Register: Formal / written — rarely used in casual speech

POUR + infinitif = In order to / To (SAME subject)
→ "Je travaille POUR réussir." (I work to succeed — same person)
→ NO subjonctif — just infinitif
→ ❌ "Je travaille pour que je réussisse." (WRONG — same subject)
→ ✅ "Je travaille pour réussir." (CORRECT)

AFIN DE + infinitif = In order to (SAME subject — formal)
→ "Afin de mieux comprendre, j'ai relu le document."
→ Same as "pour" but more formal

SAME-SUBJECT RULE (Connection to P23):
→ Same subject → POUR / AFIN DE + infinitif
→ Different subjects → POUR QUE / AFIN QUE + subjonctif

🧠 "DE SORTE QUE / DE FAÇON QUE / DE MANIÈRE QUE" — The Dual-Mood Connectors:

These connectors can take EITHER indicatif OR subjonctif:

SUBJONCTIF = PURPOSE (so that — intention):
→ "J'ai parlé doucement DE SORTE QU'il ne m'ENTENDE pas."
   = I spoke softly SO THAT he wouldn't hear me. (my intention)

INDICATIF = RESULT (so that — actual outcome):
→ "J'ai parlé doucement, DE SORTE QU'il ne m'a pas ENTENDU."
   = I spoke softly, so he didn't hear me. (what actually happened)

→ Connection to Project 28: "de sorte que" appeared as consequence connector (indicatif)
→ Now you see it can ALSO be a purpose connector (subjonctif)
→ The MOOD tells you whether it's purpose (subj) or result (indic)

🧠 "À CONDITION QUE" vs. "POURVU QUE":

À CONDITION QUE + subjonctif = On condition that / Provided that
→ Sets a SPECIFIC CONDITION for something to happen
→ "Tu peux sortir à condition que tu fasses tes devoirs."
→ Implies: IF you don't meet the condition → the result won't happen
→ Register: Neutral — used in conversation and writing

POURVU QUE + subjonctif = Provided that / Let's hope that
→ TWO USES:

USE 1 — CONDITION (= à condition que):
→ "Pourvu que tu aies ton billet, tu pourras entrer."
→ = Provided you have your ticket...

USE 2 — WISH/HOPE (standalone — very common):
→ "Pourvu qu'il fasse beau !" = Let's hope the weather is nice!
→ "Pourvu que ça marche !" = Let's hope it works!
→ "Pourvu que..." alone = an anxious HOPE
→ No consequence clause needed — the wish stands alone
→ Very common in spoken French

SAME-SUBJECT ALTERNATIVE:
→ à condition de + infinitif (same subject)
→ "Tu peux sortir à condition DE finir tes devoirs."
→ "J'accepte à condition DE recevoir une confirmation."

🧠 "SANS QUE" — Without [someone doing something]:

SANS QUE + subjonctif = Without [someone] doing [something]
→ ALWAYS subjonctif — no exceptions
→ "Il est parti SANS QUE je le SACHE." = He left without me knowing

SAME-SUBJECT ALTERNATIVE:
→ SANS + infinitif (same subject)
→ "Il est parti SANS dire au revoir." = He left without saying goodbye
→ Same-subject rule: sans + inf / Different: sans que + subj

EXAMPLES OF BOTH:
→ "Il mange SANS faire de bruit." (same subject — he eats quietly)
→ "Il mange SANS QUE personne le VOIE." (different — without anyone seeing)

🧠 "À MOINS QUE" — Unless:

À MOINS QUE + subjonctif = Unless
→ ALWAYS subjonctif — no exceptions
→ "Je viendrai, à moins qu'il ne pleuve." = I'll come, unless it rains.

NE EXPLÉTIF:
→ Like "avant que" and fear verbs (P23, P30)
→ "À moins qu'il NE pleuve" — the "ne" is expletive (optional, no negation)
→ Formal writing → include it
→ Casual speech → skip it: "À moins qu'il pleuve"

SAME-SUBJECT ALTERNATIVE:
→ À MOINS DE + infinitif
→ "Je viendrai, à moins D'ÊTRE malade." = I'll come, unless I'm sick
→ "À moins de trouver une solution..." = Unless we find a solution...

🧠 THE SUPERLATIVE / UNIQUE / RESTRICTIVE TRIGGER SYSTEM:

SUPERLATIVES (le plus, le moins, le meilleur, le pire) + QUE → SUBJONCTIF:
→ "C'est le meilleur film que j'AIE vu." (subjonctif)
→ "C'est la plus belle chose que je CONNAISSE." (subjonctif)

WHY? → A superlative is a SUBJECTIVE JUDGMENT — not an objective fact
→ "The best I've ever seen" = YOUR personal evaluation → subjonctif
→ Note: in casual speech, some speakers use indicatif — but subjonctif is the standard

UNIQUE/RESTRICTIVE WORDS + QUI/QUE → SUBJONCTIF:
→ le seul / la seule...qui/que    (the only...who/that)
→ le premier / la première...qui   (the first...who)
→ le dernier / la dernière...qui   (the last...who)
→ l'unique...qui                   (the unique...who)

NEGATIVE + QUI/QUE → SUBJONCTIF:
→ rien...qui/que       (nothing that...)
→ personne...qui       (nobody who...)
→ aucun...qui          (no one...who)

QUESTION + QUI/QUE → SUBJONCTIF (uncertainty):
→ "Y a-t-il quelqu'un qui PUISSE m'aider ?"
→ The existence of the person is UNCERTAIN → subjonctif

⚠️ NOT ALWAYS SUBJONCTIF:
→ If you're stating a FACT (not a judgment):
→ "C'est le premier qui EST arrivé." (indicatif — factual, I saw him arrive)
→ "C'est le premier qui SOIT arrivé." (subjonctif — emphasizing uniqueness)
→ Both are acceptable — subjonctif is more "correct" / literary
→ In casual speech, indicatif is common after superlatives

🧠 THE FIXED CONCESSIVE EXPRESSION SYSTEM:

QUOI QUE + subjonctif = Whatever (thing)
→ "Quoi que tu fasses..." = Whatever you do...
→ "Quoi qu'il arrive..." = Whatever happens...
→ "Quoi qu'il en soit..." = Whatever the case may be (FIXED — very common)

⚠️ DO NOT CONFUSE:
→ QUOI QUE (two words) = Whatever → SUBJONCTIF
→ QUOIQUE (one word) = Although → SUBJONCTIF (= bien que — P29)
→ "QUOI QUE tu fasses, je t'aime." = Whatever you do, I love you.
→ "QUOIQUE ce soit difficile, j'essaie." = Although it's difficult, I try.

OÙ QUE + subjonctif = Wherever
→ "Où que tu ailles..." = Wherever you go...
→ "Où que vous soyez..." = Wherever you are...

QUI QUE + subjonctif = Whoever
→ "Qui que vous soyez..." = Whoever you are...
→ "Qui que ce soit" = Whoever it is / Anyone (fixed expression)

QUEL(LE)(S) QUE + SOIT/SOIENT + noun = Whatever [noun]
→ AGREES in gender and number with the NOUN:
   → Quel que soit le problème (masc sing)
   → Quelle que soit la raison (fem sing)
   → Quels que soient les obstacles (masc plural)
   → Quelles que soient les conséquences (fem plural)
→ ALWAYS followed by SOIT/SOIENT (être in subjonctif)
→ Very common in formal speech and writing

🧠 SUBJONCTIF PASSÉ — When to Use It:

USE SUBJONCTIF PASSÉ when the action in the "que" clause
happened BEFORE the action in the main clause:

TIMELINE:
   [subj passé action]  ←  [main clause reaction]  ←  [now]
   He succeeded              I'm happy                  (present)

PRESENT REACTION to PAST EVENT:
→ "Je suis content que tu AIES RÉUSSI." (past success → present happiness)
→ "C'est dommage qu'il AIT PERDU." (past loss → present regret)

PAST REACTION to EARLIER PAST EVENT:
→ "J'étais content que tu AIES RÉUSSI." (earlier success → past happiness)

WITH CONNECTORS:
→ "Bien qu'il AIT FAIT des erreurs..." (past mistakes → despite)
→ "Sans qu'on l'AIT VU..." (past — without seeing)
→ "Avant qu'il AIT FINI..." (before he had finished)

VERSUS SUBJONCTIF PRÉSENT:
→ Subj présent = simultaneous or future: "Je veux qu'il VIENNE." (hasn't happened)
→ Subj passé = completed: "Je suis content qu'il SOIT VENU." (already happened)

AGREEMENT RULE:
→ Same as passé composé:
   → Avoir verbs: PP agrees with preceding COD (rare in subj passé)
   → Être verbs: PP agrees with subject
   → "C'est la seule erreur qu'il ait commisE." (que = COD fem → agreement)

🧠 THE MASTER DECISION FRAMEWORK — Subjonctif vs. Indicatif:

ASK YOURSELF: Is the "que" clause presenting something as...

REAL / CERTAIN / FACTUAL?          → INDICATIF
→ Je sais que... / Il est clair que... / Je pense que... (affirm)
→ Après que... / Parce que... / Quand...

DESIRED / DOUBTED / FEARED / JUDGED / POSSIBLE / NOT YET REAL? → SUBJONCTIF
→ Je veux que... / Il faut que... / Je doute que...
→ Pour que... / Avant que... / Bien que...
→ Il est possible que... / C'est dommage que...

SPECIAL CASE — Existence uncertain:
→ "Je cherche quelqu'un qui SACHE parler chinois." (subj — maybe nobody exists)
→ "Je connais quelqu'un qui SAIT parler chinois." (indic — the person exists)
→ SEARCHING for something uncertain → subjonctif
→ STATING something known → indicatif

THE EXCEPTIONS TO MEMORIZE:
→ ESPÉRER QUE → always INDICATIF (despite being a wish)
→ APRÈS QUE → traditionally INDICATIF (despite "avant que" being subj)
→ SI → NEVER subjonctif (P18)
→ Affirmative PENSER/CROIRE/ÊTRE SÛR → INDICATIF
→ Negative PENSER/CROIRE/ÊTRE SÛR → SUBJONCTIF
