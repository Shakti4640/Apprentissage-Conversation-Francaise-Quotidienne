🟡 PROJECT 16: "What Will Happen Someday"

You're talking about future career plans, predictions, and promises — the formal future

🎯 THE HYPOTHETICAL PROBLEM

→ You've been in France for three months now — your French has improved dramatically
→ Your language school organizes a career workshop — students discuss their professional futures
→ The career counselor asks: "Où est-ce que vous vous voyez dans cinq ans ?" (Where do you see yourself in five years?)
→ Students around you start answering:
→ → "Je travaillerai dans une entreprise internationale." (I will work in an international company.)
→ → "J'habiterai à l'étranger." (I will live abroad.)
→ → "Je serai médecin." (I will be a doctor.)
→ You try to answer using futur proche (Project 12):
→ → "Je vais travailler dans une entreprise." — technically correct, but sounds too casual and immediate for a five-year plan
→ → "Je vais être médecin." — sounds like you're about to become a doctor RIGHT NOW
→ The counselor gently corrects: "En français formel, on utilise le futur simple pour les projets à long terme."
→ You also notice the futur simple in:
→ → News broadcasts: "Le président parlera demain." (The president will speak tomorrow.)
→ → Weather forecasts: "Il pleuvra dans l'après-midi." (It will rain in the afternoon.)
→ → Promises: "Je te le promets, je reviendrai." (I promise you, I will come back.)
→ → Formal instructions: "Vous enverrez le document avant vendredi." (You will send the document before Friday.)
→ → Predictions: "Un jour, tu comprendras." (One day, you will understand.)

The core realization:
→ French has TWO future tenses — and they're NOT interchangeable in all situations:

TenseRegisterTime FrameFeeling
Futur proche (Project 12)Casual / SpokenNear future / ImmediatePlans, intentions, what's about to happen
Futur simple (Project 16)Formal / Written / SpokenNear OR distant futurePredictions, promises, formal plans, certainty

→ In daily conversation, futur proche dominates (~80% of future expressions)
→ But futur simple is ESSENTIAL for:
→ → Formal speech, professional contexts, writing
→ → Promises, predictions, philosophical statements
→ → After certain conjunctions (quand, dès que, lorsque — a rule English doesn't have)
→ → Sounding educated and professional

🔍 HOW TO APPROACH IT

→ The futur simple is built differently from everything you've learned so far
→ It does NOT use an auxiliary verb (no avoir, no être, no aller)
→ It's a single-word tense — the future meaning is built INTO the verb itself
→ The formula:

INFINITIF (full verb) + FUTURE ENDINGS

→ You literally take the infinitif and ADD endings to it
→ The endings are the SAME for ALL verbs — no exceptions
→ The only thing that varies is the STEM — regular verbs use the full infinitif, irregular verbs have special stems

🔧 HOW CAN WE SOLVE IT

🧠 NOTICE SOMETHING?
→ These endings look exactly like the present tense of AVOIR:
→ → j'ai, tu as, il a, nous avons, vous avez, ils ont
→ This is NOT a coincidence — historically, the futur simple was built from the infinitif + avoir
→ → "Parler + ai" → "Parlerai" = "I will speak" (literally: "to speak I have")
→ This makes them easier to remember — you already know avoir!

🧠 CRITICAL FRENCH RULE — FUTUR AFTER "QUAND":

→ In English: "When I arrive, I will call you." → present tense after "when"
→ In French: "Quand j'arriverai, je t'appellerai." → FUTURE tense after "quand"!

LanguageAfter "when/quand"Main clause
EnglishPresent tenseFuture
FrenchFUTUR SIMPLEFutur simple

→ This is one of the BIGGEST differences between English and French grammar
→ English speakers instinctively put present tense after "quand" → WRONG in French
→ → ❌ Quand j'arrive, je t'appellerai. (English logic — wrong in French)
→ → ✅ Quand j'arriverai, je t'appellerai. (French logic — future after quand)

🧠 This rule applies to ALL time conjunctions referring to the future:

ConjunctionMeaningExample
QuandWhenQuand je serai grand...
LorsqueWhen (formal)Lorsque tu arriveras...
Dès queAs soon asDès que j'aurai le temps...
Aussitôt queAs soon asAussitôt que tu pourras...
Tant queAs long asTant que je vivrai...

→ After ALL of these → use futur simple (or futur antérieur — much later)
→ This rule does NOT apply to "si" (if) → recall from Project 12: si + present, NOT future
→ → ✅ Si tu viens, on ira au cinéma. (si + present → futur)
→ → ✅ Quand tu viendras, on ira au cinéma. (quand + futur → futur)

❌ WHAT SHOULD NOT DO

→ ❌ Do NOT use present tense after "quand" for future events → "Quand j'arrive, je t'appelle" (for future meaning) → "Quand j'arriverai, je t'appellerai"
→ ❌ Do NOT use futur simple after "si" → "Si je serai riche..." → "Si je suis riche..." (present after si)
→ ❌ Do NOT add future endings to the past participle → "J'ai mangérai" → "Je mangerai"
→ ❌ Do NOT confuse futur simple with conditionnel endings → je parlerai (future) vs. je parlerais (conditional — Project 17)
→ ❌ Do NOT use irregular present stems for futur → "Je fairai" → "Je ferai" (stem = fer-)
→ ❌ Do NOT forget to drop the -e from -RE infinitifs → "J'attendreai" → "J'attendrai"
→ ❌ Do NOT use futur simple for IMMEDIATE events in casual speech → "Watch out, you'll fall!" = "Tu vas tomber !" (futur proche — more urgent)
→ ❌ Do NOT use futur proche in formal writing when futur simple is expected → professional emails, reports, announcements
→ ❌ Do NOT confuse "je voudrai" (future — I will want) with "je voudrais" (conditional — I would like)

Project 16 → Futur simple = formal/certain future

The futur simple structure:

INFINITIF (or irregular stem) + ENDINGS (-ai, -as, -a, -ons, -ez, -ont)

Three critical rules:

  1. After quand/dès que/lorsque → MUST use futur simple (not present, not futur proche)
  2. After si → NEVER use futur (use present — Projects 12/18)
  3. Endings are the same for ALL verbs — only stems vary

Five essential irregular stems: ser- (être), aur- (avoir), ir- (aller), fer- (faire), viendr- (venir)

Futur proche (P12) vs. Futur simple (P16):
→ Futur proche = casual, immediate, spoken
→ Futur simple = formal, distant, written, promises, after quand
→ Both are essential — they complement each other

But all your future statements are FLAT — you can say "I will" but not "I would."
→ "I would like a coffee" / "Could you help me?" / "If I were rich..."
→ These require the conditionnel — the tense of politeness, wishes, and imagination.

🌉 BRIDGE TO PROJECT 17

You can now say "I will work" and "When I arrive, I will call you"
But at a formal dinner, someone asks: "Que voudriez-vous boire ?" (What would you like to drink?)
You want to answer politely — not "Je veux" (too direct — Project 6) and not "Je voudrai" (wrong tense — that's future)
→ You need "Je voudrais" — the conditionnel
→ The conditionnel is used for:
→ → Polite requests ("I would like...")
→ → Imagined situations ("If I were rich, I would travel")
→ → Gentle suggestions ("You could try this restaurant")
→ → Uncertain information ("According to reports, there would be...")
→ Project 17: "Could You, Would You" unlocks the conditionnel présent
→ It uses the SAME irregular stems as the futur simple + DIFFERENT endings (imparfait endings!)
→ This is where your French starts sounding truly SOPHISTICATED.
