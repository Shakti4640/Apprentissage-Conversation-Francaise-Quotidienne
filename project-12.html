🔵 PROJECT 12: "What's About to Happen"

You're making plans for tonight, this weekend, right now

🎯 THE HYPOTHETICAL PROBLEM

→ It's Friday afternoon at the language school — the last class of the week just ended
→ Everyone is excited — the weekend is here
→ Your classmates start asking each other:
→ → "Tu vas faire quoi ce soir ?" (What are you going to do tonight?)
→ → "On va manger où demain ?" (Where are we going to eat tomorrow?)
→ → "Vous allez sortir ce week-end ?" (Are you going to go out this weekend?)
→ You understand every question — but you can only answer in present tense
→ "Ce soir, je mange chez moi." — This works, but sounds like you're describing a habit, not a plan
→ "Demain, je visite le Louvre." — Technically understandable, but missing the intention
→ You need to express:
→ → "I'm going to eat at home tonight."
→ → "I'm going to visit the Louvre tomorrow."
→ → "We're going to meet at 8."
→ → "She's about to leave."
→ → "It's going to rain."
→ → "I'm not going to work this weekend."

The core realization:
→ French has TWO future tenses:
→ → Futur proche (near future) = aller + infinitif → "I'm going to..."
→ → Futur simple (simple future) = special verb endings → "I will..." (Project 16)
→ In daily spoken French, the futur proche is used far more often than the futur simple
→ It covers plans, intentions, predictions, things about to happen, and even some distant future events in casual speech
→ And the best news: you already know everything you need to build it

🔍 HOW TO APPROACH IT

→ The futur proche structure:

SUBJECT + ALLER (present tense) + INFINITIF + rest

→ You already know:
→ → Aller in present tense (Project 2): je vais, tu vas, il va, nous allons, vous allez, ils vont
→ → Infinitif (Project 6): the dictionary form of the verb — manger, partir, dormir, faire...
→ → The modal pattern (Project 6): conjugated verb + infinitif → je veux manger, je peux partir
→ The futur proche follows the exact same pattern as modals — just replace the modal with aller

Modal (Project 6)Futur Proche (Project 12)
Je veux manger. (I want to eat.)Je vais manger. (I'm going to eat.)
Je peux partir. (I can leave.)Je vais partir. (I'm going to leave.)
Je dois travailler. (I must work.)Je vais travailler. (I'm going to work.)

→ Same skeleton — different control verb
→ If you mastered Project 6 → Project 12 is almost free

🔧 HOW CAN WE SOLVE IT

Step 1 — Recall ALLER in Present Tense (Project 2)

SubjectAller Form
Jevais
Tuvas
Il / Elle / Onva
Nousallons
Vousallez
Ils / Ellesvont

→ Already memorized from Project 2
→ In Project 2/7, aller was used for MOVEMENT: "Je vais au bureau" (I go to the office)
→ In Project 12, aller is used for FUTURE: "Je vais manger" (I'm going to eat)
→ The difference: what follows aller
→ → Aller + PLACE = movement (going somewhere)
→ → Aller + INFINITIF = future (going to do something)

Step 2 — The Complete Futur Proche Formula

SUBJECT + ALLER (conjugated) + VERB (infinitif) + rest of sentence

→ ONLY aller conjugates — the action verb stays in infinitif
→ Same rule as modals (Project 6): never conjugate the second verb
→ ✅ Je vais manger. (correct)
→ ❌ Je vais mange. (wrong — must be infinitif)
→ ❌ Je vais mangé. (wrong — that's a past participle, not infinitif)

🧠 CRITICAL CONFUSION TO AVOID:
→ "Mangé" (past participle — Project 9) looks similar to "manger" (infinitif)
→ In speech, they sound IDENTICAL: "man-zhay"
→ In writing:
→ → J'ai mangé = passé composé (I ate)
→ → Je vais manger = futur proche (I'm going to eat)
→ → The spelling difference matters in writing!

Step 3 — Negation in Futur Proche

SUBJECT + NE + VAIS/VAS/VA... + PAS + INFINITIF + rest

→ Same rule as all negation with conjugated verbs (Project 5, Project 6):
→ The negation sandwich wraps around the conjugated verb (aller)
→ The infinitif stays OUTSIDE the sandwich

→ ✅ Je ne vais pas manger. (I'm not going to eat.)
→ ❌ Je vais ne pas manger. (different meaning — "I'm going to not eat" — intentional non-eating)

→ Spoken French (Project 5 reminder): drop "ne"
→ Written: Je ne vais pas manger.
→ Spoken: Je vais pas manger.

Step 4 — When to Use Futur Proche

→ Futur simple (Project 16) is more formal/written/distant

✅ WHAT SHOULD DO

→ Use futur proche as your DEFAULT future tense — it's the most common in spoken French
→ Keep the infinitif after aller — never conjugate the second verb
→ Combine with time markers from Project 8 — ce soir, demain, la semaine prochaine, dans + duration
→ Learn the "dans + duration" time marker — dans cinq minutes, dans une heure, dans deux jours
→ Practice negation — ne + aller + pas + infinitif (same pattern as all other negation)
→ Use "ça va + infinitif" freely — it's the most casual and natural way to predict things
→ Stack aller + modal + infinitif for complex futures — je vais devoir partir, on va pouvoir manger
→ Connect to Project 2 — aller conjugation is the same (je vais, tu vas, il va...)
→ Connect to Project 6 — same modal structure: conjugated verb + infinitif
→ Connect to Project 8 — time markers are the same, just shifted to future context

❌ WHAT SHOULD NOT DO

→ ❌ Do NOT conjugate the second verb → "Je vais mange" → "Je vais manger"
→ ❌ Do NOT confuse infinitif (-er) with past participle (-é) in writing → "Je vais manger" ≠ "J'ai mangé"
→ ❌ Do NOT put aller in the future to make futur proche → "J'irai manger" = futur simple (Project 16), not futur proche
→ ❌ Do NOT use futur proche after "si" (if) → "Si je vais manger..." → "Si je mange..." (present after si)
→ ❌ Do NOT use futur proche for habitual present actions → "I go to school every day" = present tense, not futur proche
→ ❌ Do NOT say "Je vais aller aller" (triple aller) → "Je vais aller au cinéma" is the maximum (aller + aller)
→ ❌ Do NOT place negation around the infinitif → "Je vais ne pas manger" has a different meaning (intentional not-eating) vs. "Je ne vais pas manger" (plan negation)
→ ❌ Do NOT forget that "aller + place" ≠ futur proche → "Je vais au bureau" (present, movement) vs. "Je vais travailler" (future, plan)

→ Rien goes BEFORE the infinitif / Personne goes AFTER the infinitif

🚫 WHEN NOT TO USE

→ Do NOT use futur proche for formal predictions about distant future → use futur simple (Project 16)
→ → "In 50 years, the climate will change" → futur simple territory
→ Do NOT use futur proche in the "si" clause of conditional sentences → always present tense after si
→ Do NOT use futur proche for past intentions → "I was going to eat" = "J'allais manger" (imparfait of aller + infinitif — not covered yet)
→ Do NOT use futur proche for habitual actions → "I eat every day" = present tense, not futur proche
→ Do NOT use futur proche when something is happening right now → use present: "Je mange" (I'm eating now) vs. "Je vais manger" (I'm going to eat later)

🧠 PREVIEW — "J'ALLAIS + infinitif" (past intention):
→ If you put aller in imparfait instead of present → you get past intention
→ "J'allais partir" = I was going to leave / I was about to leave
→ "On allait manger quand le téléphone a sonné" = We were about to eat when the phone rang
→ This is the imparfait of aller + infinitif → "futur proche in the past"
→ Very useful but not the focus of this project — just be aware it exists

The futur proche structure:

Subject + ALLER (present) + INFINITIF + rest
