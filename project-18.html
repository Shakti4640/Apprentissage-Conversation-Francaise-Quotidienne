🟡 PROJECT 18: "If This, Then That — Part 1"

Someone asks "What if it rains?" — you answer with a real plan

🎯 THE HYPOTHETICAL PROBLEM

→ You live in France now — life is full of conditions
→ Your friend says: "What do we do this weekend?"
→ Your answer depends on weather, money, time, mood, availability
→ Your boss says: "Can you finish by Friday?"
→ Your answer depends on workload, help, deadlines
→ Your partner says: "Where should we go for vacation?"
→ Your answer depends on budget, dates, preferences
→ Every plan, every decision, every negotiation in real life is conditional
→ You need to say: "IF this happens → THEN that will happen"
→ French has a strict system for this — break the rules and you sound broken

The core realization:
→ In Project 16, you learned to say what WILL happen (futur simple)
→ In Project 17, you learned to say what WOULD happen (conditionnel)
→ Now you must learn: what TRIGGERS which future — and the trigger word is SI
→ "Si" is the gatekeeper — it controls which tense is allowed after it and which tense follows in the result

🔍 HOW TO APPROACH IT

→ Every conditional sentence has TWO parts:

SI-CLAUSE (the condition) → RESULT CLAUSE (the consequence)

→ Think of it like a door and a room:
→ The SI-clause is the door — it sets the condition
→ The result clause is the room — it tells what happens if you walk through

→ In Project 18 (Part 1), we deal with REAL / LIKELY conditions
→ These are things that CAN actually happen — not fantasy

→ Three combinations are possible at this level:

SI-Clause TenseResult Clause TenseMeaning
Si + présent→ futur simpleIf X happens, Y will happen
Si + présent→ présentIf X happens, Y (always) happens
Si + présent→ impératifIf X happens, do Y!

🔧 HOW CAN WE SOLVE IT

THE GOLDEN RULE OF FRENCH SI-CLAUSES

❌ NEVER put FUTUR or CONDITIONNEL directly after "SI"
✅ SI always takes PRÉSENT (in Part 1), IMPARFAIT (Part 2 — Project 33), or PLUS-QUE-PARFAIT (Part 3 — Project 34)

→ This is the single most violated rule by French learners
→ In English you can say "If I will go..." — in French this is IMPOSSIBLE
→ ❌ Si je serai là → completely wrong
→ ✅ Si je suis là → correct

Memorize this now — it never changes across all 3 si-clause projects

🔑 KEY TAKEAWAY

The si-clause is the CONTROL CENTER of French conditional thinking.

→ In Project 16, you learned to make statements about the future (futur simple)
→ In Project 17, you learned to make softened/imagined statements (conditionnel)
→ Now in Project 18, you learned WHAT TRIGGERS THEM — the si-clause

The rule is absolute and permanent:
→ Si + PRÉSENT → futur / présent / impératif = REAL possibility
→ Si + IMPARFAIT → conditionnel = UNREAL present (Project 33)
→ Si + PLUS-QUE-PARFAIT → conditionnel passé = UNREAL past (Project 34)

You now own Part 1 of 3 — the real, likely, possible conditional system
Parts 2 and 3 will unlock imagination and regret — but they follow the SAME skeleton
